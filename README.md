# Aho View

A simple and fast image viewer.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/wxlin1983/aho-view2.git
    cd aho-view2
    ```

2.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To run the application, execute the `main.py` script:

```bash
cd src
python main.py
```

You can then open a directory of images using the "File" -> "Open Directory..." menu or by dragging and dropping a directory onto the window.
